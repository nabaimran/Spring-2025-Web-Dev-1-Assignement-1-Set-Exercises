HTML & CSS Quick Reference Guide

HTML Tags

Below are some commonly used HTML tags:

1. Heading Tags

Used to define headings from h1 (largest) to h6 (smallest).

<h1>This is a Heading</h1>

2. Paragraph Tag

Defines a paragraph.

<p>This is a paragraph.</p>

3. Anchor Tag

Used to create hyperlinks.

<a href="https://example.com">Visit Example</a>

4. Image Tag

Displays an image.

<img src="image.jpg" alt="Description">

5. List Tags

Used to create ordered and unordered lists.


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

CSS Properties

Below are some commonly used CSS properties:

1. Background Color

body { background-color: lightgray; }

2. Font Size

p { font-size: 16px; }

3. Text Alignment

h1 { text-align: center; }

4. Padding & Margin

div { margin: 10px; padding: 5px; }

5. Border

p { border: 1px solid black; }
